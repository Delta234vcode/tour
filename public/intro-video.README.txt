CHAIKA EVENTS — інтро-відео

Щоб відео завантажувалось миттєво (з диска / HTTP-кешу того ж сайту):
  1. Перейменуйте ваш MP4 на: intro.mp4
  2. Покладіть файл у цю папку public/

Після build файл опиниться як /intro.mp4 — браузер агресивніше кешує, ніж зовнішні CDN.

Опційно в .env.local:
  VITE_INTRO_VIDEO_URL=https://ваш-домен/intro.mp4
